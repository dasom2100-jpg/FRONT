
fadeIn(), fadeOut()함수 : 
불투명도 조절. speed는 fast, normal(400), slow 또는 0 이상의 숫자(밀리세컨드 단위)를 
설정할 수 있다. 
애니메이션이 끝난 후에 실행할 함수를 지정할 수 있다.

fadeIn( [시간 ms] [, 속도함수값 ] [, 콜백함수 ] ) 
- 선택한 요소를 서서히 나타나게 하는 기능.
fadeOut( [시간 ms] [, 속도함수값 ] [, 콜백함수 ] ) 
- 선택한 요소를 서서히 사라지게 하는 기능.

시간(ms) : fast(대략: 200) | slow(대략: 600) | 숫자값 입력(1/1000초).
입력값이 없을 경우 기본값 400으로 설정됨.

easing(모양-시간당 속도 함수값) : 'swing'(기본값) | 'linear'
swing : 시작과 끝은 느리게, 중간은 빠른 속도로
linear : 일정한 속도로

콜백함수 : 실행할 함수
===================================================

$.animate()  함수: 
- CSS로는 표현하기 애매한 동적 인터랙션등을 제어하기 위한 함수
- 다양한 동작효과를 적용할 수 있다.
=> 동적 인터랙션이란 : 사용자의 행동에 따라 화면이 즉시 반응하면서 계속 바뀌는 동작을 말함
형식: animate({스타일속성}, 적용시간 | 속도함수 | 콜백함수) 
적용시간(ms), 속도함수('swing', 'linear' - 반드시 문자열로 표기)
 "linear" : 처음부터 끝까지 같은 속도
 "swing" : 처음과 끝은 느리고, 중간은 빠름(기본값)

스타일속성명은 -를 제외하고, 카멜기법으로 표기, 반드시 숫자값으로 계산 가능한 속성만 가능함.
padding처럼 단축속성으로 수치값이 여러 개 동시 기술되는 것들은 불가

** stop()함수 : 동일한 누적이벤트를 멈추는 기능
animate는 애니메이션기능을 줄 세우는 성격이 있어 스크롤 기능처럼 지속적으로 발생되는 이벤트는
stop()을 해주지 않으면 이벤트가 누적되어 이상하게 동작이 될 수 있다. 
따라서, 동일한 이벤트가 누적된 것이 있으면 멈추고 현재 발생된 이벤트만을 적용하도록 해주어야 하는데,
그때 사용하는 것이 stop()함수이다.

** $.animate() 함수에서 문자열값을 가진 속성을 사용하거나 문자열값을 가진 속성에 대한 동작을 지원받으려면 해당 기능을 제공해주는 jquery ui 라이브러리를 추가해야 한다.

현재 파일이 의도한 정확한 파일이 맞는지를 브라우저가 검증하도록 하는 일종의 인증코드와 같은 속성임. crossorigin 속성과 동시에 짝으로 사용됨
=> 현재 내가 사용하는 파일의 integrity값과 CDN서버의 파일을 내려받아서 해당 파일의 매칭해시코드 값을 비교함.
=> CDN 서버가 해킹되면, 해킹 공격으로 CDN파일이 변경될 수 있음. 그런데 integrity속성이 있으면 값을 비교하여, 일치하면 실행, 조금이라도 변경되면 실행안됨. integrity속성이 없는 경우 혹 파일에 "악성코드"가 존재할 경우 그대로 실행되므로 문제가 될 수 있음
=> 교육용에서는 integrity가 없어도 괜찮지만 실서비스 / 기업 / 금융 / 공공 사이트에선 위험할 수 있으므로 필수(권장)

crossorigin="anonymous"
브라우저는 교차 출처 리소스를 가져올 때(CORS) 보안 정책을 적용하는데 해당 교차 출처 리소스를 가져올 때 필요한 속성임
anonymous의 의미
- 쿠키, 인증 정보 없이 요청
- 가장 안전하고 일반적인 방식
- CDN 라이브러리에는 거의 무조건 이 속성을 사용해야 함.
** 만약 integrity는 있는데 crossorigin이 없으면 브라우저에 따라 SRI 검증 실패 및 특히 폰트, 스크립트에서 문제가 생길 수 있음
** CORS(= Cross-Origin Resource Sharing) : 본래의 도메인과 서로 다른 도메인간의 자원 공유(주고받기)를 위한 정책
