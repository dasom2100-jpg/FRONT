jquery를 사용하는 방법 (외부스크립트로 이용가능)

1. 네트워크 방식
=> jquery.com페이지로 접속
=> Download 메뉴 클릭
=> 스크롤 내려서 "jQuery CDN" 내용의 "Other CDNs"내용까지 스크롤로 이동
-------------------
Google CDN
Microsoft CDN
CDNJS CDN
jsDelivr CDN
-------------------
위의 4개 중에서 선택하여  CDN Url추가하면 됨
형식 :  <script src="CDN url값"></script>
예: <script src="https://ajax.aspnetcdn.com/ajax/jQuery/jquery-3.7.0.min.js"></script>

2. 다운로드 방식
=> 위의 1번 "Other CDNs"내용까지 스크롤로 이동하는 것까지 동일하게 한 뒤
=> CDN 4개 중에 한 개 선택하여 https:// 나 http:// 로 시작하는 url주소를 복사
=> 복사한 url주소를 크롬브라우저의 주소창에 붙여넣기한 뒤 엔터로 이동
=> 브라우저에 나오는 스크립트내용안에서 마우스 우버튼 클릭 => 다른이름으로 저장
=> 원하는 경로에 원하는 파일명으로 반드시 압축파일은 min.js로 비압축파일은 .js로 
저장할 것

형식 :  <script src="다운로드받은 파일 경로"></script>
<script src="js/3.7.1/jquery.js"></script>
