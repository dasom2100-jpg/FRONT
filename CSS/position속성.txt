position : 엘리먼트(태그)의 위치를 조정하는 속성
- static (디폴트) : 기본값으로 top, bottom, right, left속성, z-index속성이 적용이 안됨. 
**static을 제외하고는 다른 속성값은  top, bottom, right, left속성, z-index속성 적용 가능

- relative : 상대적으로 위치를 조정할 수 있는 속성
- absolute : 절대적으로 위치 조정. float와 비슷하게 위에 둥둥 떠다니는 속성이며(단, 종속되지 않음), 부모위치를 relative를 이용하여 조절 가능함. 
- fixed : 절대적으로 위치를 고정하는 속성. 부모를 body만 인식함.
- sticky : 원하는 지점에서 스티커를 붙여놓은 듯 붙이는 곳을 지정할 수 있는 속성. 감싸고 있는 태그를 부모로 인식하므로 원하는 지점에 붙여놓을 수 있는 효과를 원하는 경우에는 body의 자식으로 존재해야 한다.
