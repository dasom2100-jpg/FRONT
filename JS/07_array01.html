Array(배열)이란:

배열은 같은 타입의 여러 데이터를 하나의 변수에 저장할 수 있는 자료구조입니다.

배열 선언 방법:

    // 1. 배열 리터럴 방식
    var fruits = ["apple", "banana", "cherry"];
    // 2. Array 생성자 방식
    var numbers = new Array(1, 2, 3, 4, 5);

배열 요소 접근:

    var firstFruit = fruits[0]; // "apple"
    var secondNumber = numbers[1]; // 2

배열 메서드 예시:

    // push(): 배열 끝에 요소 추가
    fruits.push("date"); // ["apple", "banana", "cherry", "date"]
        
    // pop(): 배열 끝의 요소 제거
    var lastFruit = fruits.pop(); // "date", fruits는 ["apple", "banana", "cherry"]     
    // length: 배열의 길이 반환
    var fruitCount = fruits.length; // 3

배열 순회:

    for (var i = 0; i < fruits.length; i++) {   
        console.log(fruits[i]);
    }
