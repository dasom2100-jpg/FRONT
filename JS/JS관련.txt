자바스크립트의 변수(=객체) : 
1. 값 저장 + 함수, 속성 모두 사용가능
2. 자료형 제한이 없다.
3. 선언만 하는 경우 초기값 : undefinded

선언형식 :  
키워드 변수명;  => 예)  let a;
키워드 변수명 = 값;   예)  const a = 5;
키워드 변수명, 변수명, ...;   
키워드 변수명= 값, 변수명=값,...;

키워드 : 전역, 지역, 상수를 구분해주는 예약어
전역 : var (JS=ES5) : 
전역적으로 사용할 수 있는 변수, 재선언, 재할당 가능

지역 : let (ES6) : 
선언된 범위내에서만 사용할 수 있는 변수.
재할당만 가능. 
재선언 불가(단, 범위가 다른 지역내에서는 재선언 가능)


상수 : const (ES6)
선언된 범위내에서만 사용할 수 있으며, 값이 고정된 변경할 수 없는 상수.
재할당, 재선언 모두 불가(단, 범위가 다른 지역내에서는 재선언 가능)
반드시 선언과 동시에 초기화(값 할당)까지 이루어져야 한다.

1. 스크립트의 종류 
인라인스크립트 : html의 시작태그에 이벤트함수속성을 사용하여 기술하는 방식
내부스크립트 : html의 script태그안에 기술하는 방식  <script></script>
외부스크립트 : html의 script태그안에 src속성을 사용하여 외부 스크립트 파일을 import하여 사용하는 방식   <script src="스크립트파일형식"></script>

2. 스크립트의 우선순위
인라인,내부,외부 상관없이 무조건 해당되는 것은 모두 실행됨.
단, 동일한 기능은 기술된 순서에 따라 가장 마지막에 기술된 것으로 덮어쓰기 됨.
