선언적함수 : 함수의 형식을 정식으로 지켜서 선언한 함수이면서, 단독으로 호출하여 사용이 가능한 함수
호이스팅 가능
function 함수명(){

}

익명함수 : 이름이 없이 선언이되는 함수이며, 단독으로 사용 불가한 함수
호이스팅 불가
형식 : function( [매개변수들...] ){ 자바스크립트 코드들...; }
익명함수 사용방법
1. 변수의 리터럴로 존재 :
변수를 선언하여 해당 값으로 익명함수를 선언하는 방식
- 장점 : 여러 곳에 매개 인자로 보내어 사용하기가 용이함
형식 : 
let 변수명= function( [매개변수들...] ){ 자바스크립트 코드들...; } ;
호출문형식 : 변수명([인자값]);

2. 함수호출문의 인자값으로 존재 : 
함수호출문에 인자값으로 선언하는 방식
- 장점 : 
객체를 호출문안에 만들기 때문에 메모리 낭비를 줄일 수 있다.
함수가 호출되면서 자동으로 실행된다.
형식 :
객체명.함수호출문( function( [매개변수들...] ){ 자바스크립트 코드들...; } );


전역함수와 지역함수(내장함수)
- 전역함수 : 
해당 함수가 포함되어 있는 스크립트를 가져와서 사용하는 웹문서내에 전역적으로 사용 가능함.
직접 호출이 가능함

- 지역함수 : 
함수내부안에 존재하는 중첩되어있는 내장함수를 말함.
외부에서 직접호출 불가
- 내부안에서만 호출하여 사용하거나 외부에서 호출하는 경우는 반드시 감싸고 있는 부모 함수의 리턴값을 통하여 사용가능함.
